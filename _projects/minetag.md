# Gameplay
{: .align-center}

You have a lasergun that oneshots your enemies.
Shift points ratio by shooting enemy team players, game ends when one of the teams reaches 100 : 0
{: .align-center}

# Special Items
{: .align-center}

Once every 30-60 seconds one or two Lootboxes spawns in a random spot on the map. A Lootbox contains one of the three special items
- Grenade. When explodes, creates a particle cloud that kills enemy team players but keeps teammates alive. Has a secret special feature.
- Remote #1 [​BLOW up enemy players]. Creates an area below every enemy player that explodes shorly.
- Remote #2 [​GLOW up enemy players]. Glows up enemy players for 10 seconds.

Sometimes Super Armor can spawn in a random spot on the map. When equiped, gives player 3 extra "health points".

# How to play

1. (For the host) Download the map
2. (For everyone) Download the resourepack
3. (For the host) Set up the server
4. (For everyone) Join the host's server
5. Have fun!
