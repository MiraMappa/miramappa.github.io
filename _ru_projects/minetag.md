# Геймплей
{: .align-center}

У вас есть лазерная пушка, которая убивает врагов с одного выстрела
Сдвигайте соотношение очков, стреляя во вражеских игроков, игра окончится когда одна из команд достигнет 100 : 0
{: .align-center}

# Особые предметы
{: .align-center}

Каждые 30-60 секунд один или два ящика появляется в случайной точке на карте. Ящик содержит один из трёх особых предметов
- Граната. После взрыва создаёт облако частиц, которое убивает вражеских игроков, но не трогает союзников. Имеет особый секретный функционал
- Пульт №1 [ПОДОРВАТЬ врагов]. Создаёт область под каждым вражеским игроком, которая взрывается через короткий отрезок времени.
- Пульт №2 [ПОДСВЕТИТЬ врагов]. Подсвечивает вражеских игроков на 10 секунд

Иногда Супер Броня может появиться в случайной точке на карте. При экипировании даёт игроку 3 дополнительных "очка здоровья".

# Как играть

1. (Для хоста) Скачайте карту
2. (Для всех) Скачайте ресурспак
3. (Для хоста) Настройте сервер
4. (Для всех) Присоединитесь к серверу хоста
5. Веселитесь!

Музыка, использованная на карте:

- Toby Fox - Smart Race
- Toby Fox - Checker Dance
- Toby Fox - Pandora Palace
- Toby Fox - Knock You Down
